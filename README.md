## INSTALLATION :

### Termux
* `pkg update -y`
* `pkg upgrade -y`
* `pkg install git -y`
* `git clone https://github.com/Alygnt/Clifty`
* `cd Clifty`
* `bash clifty.sh`

### Kali Linux
* `sudo apt update -y`
* `sudo apt upgrade -y`
* `sudo apt install git -y`

* `sudo bash clifty.sh`
